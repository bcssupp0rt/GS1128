GS1 128 Bar Code

GS1-128, formerly known as UCC/EAN-128 Symbology, is a variant of the Code 128 Symbology. The UCC EAN System was established by Uniform Code Council (UCC) in 1973. In 1977, the European Article Numbering Association (EAN International) designed a compatible system for use in Europe. The pattern of using FNC1 right after the start character is reserved exclusively for GS1128.
